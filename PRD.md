# PRD: Krishna Academy Digital Platform

## 🎯 Objective
Digitalizing the offline + online coaching ecosystem. Providing a centralized system for students to track their academic journey and for admins to manage the institution efficiently.

## 👥 User Roles
### 👨🎓 Student
- **Dashboard**: High-level summary of status.
- **Attendance**: Detailed subject-wise and monthly tracking.
*   **Fees**: Payment status and transaction history.
- **Lectures**: Daily/Weekly schedule and pre-recorded content.
- **Courses**: Interactive video learning.

### 🧑💼 Admin
- **Student Management**: Full CRUD operations on student records.
- **Academic Control**: Mark attendance, schedule lectures, upload courses.
- **Financial Control**: Manage and update fee records.

## 🌐 Public Website Features
- **Home**: 3D premium hero section with video background.
- **Courses**: Searchable listing of available academic programs.
- **About/Faculty**: Institutional heritage and team details.
- **Conversion**: Admissions CTA and Student Portal login.

## 🛠️ Tech Stack
- **Frontend**: Next.js 16 (App Router) + Framer Motion.
- **UI**: Tailwind CSS (Monumero Theme).
- **Backend**: Supabase (PostgreSQL + Auth + Storage).
- **Payments**: Razorpay (HMAC SHA-256 Verification).
- **Hosting**: Vercel (Subdomain rewrites).

## 🗄️ Core Database
- **Profiles**: Auth-linked user metadata.
- **Attendance**: Daily status logs.
- **Fees**: Transactional and pending fee records.
- **Lectures**: Time-slotted academic sessions.

## 🚀 Roadmap
- **V2**: Live interactive classes.
- **V3**: AI-powered student doubt resolution tutor.
- **V4**: Native Mobile Application.
